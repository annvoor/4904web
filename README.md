# 4904 Training Site

## Running on Web

Go to [4904training.netlify.app](4904training.netlify.app)

## Running on LocalHost
Once you've cloned your repo:

```bash
# Make sure you have npm and vite installed
# Using homebrew:
brew install node
brew install vite

# install the dependencies
npm install

#open the project!
npm run dev -- --open
```
